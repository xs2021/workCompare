

## 一 . 准备花名册

格式要求：

- 存储方式：txt文本文档

- 姓名格式：一行一个

- **倘若姓名有这样子的：**

  一个人的姓名刚好是另一个人的前几个字，这种需要把名字较长的放到【花名册】的前边
